Web Aplikasi Check Room Mahasiswa Unai

Februari 27, 2022

Latar Belakang

Para mahasiswa dan mahasiswi UNAI yg tinggal di asrama di asrama di wajibkan sudah ada di kamar pada jam 09 : 30, para monitor di setiap hall asrama melakukan pengecekan ke setiap kamar untuk memastikan semua mahasiswa sudah ada di kamar masing - masing. Para monitor melakukannya masih menggunakan cara manual dan dikarenakan ada tugas project untuk membuat web aplikasi jadi oleh karena itu saya ingin membuat web aplikasi Check Room mahasiswa UNAI.

Fungsi Aplikasi

  • Login memasukan nama dan password
  • Memilih asrama
  • Menampilkan tabel kamar Mahasiswa dan button check
  • Menampilkan tabel data mahasiswa yg ada di kamar tersebut
  • Checkbox hadir, dan tidak hadir
  • Button simpan
